# Twixter--A-clone-of-Twitter


React Web App Clone (WAP Group Project)

## Project Overview

This project is a **React-based clone** of a popular web application (such as YouTube, Instagram, or a similar platform). It showcases modern frontend development using React with a focus on interactivity, reusable components, and responsive design.

The application uses **static data or JSON Server** to simulate backend functionality.

---

Objectives
- Build a functional frontend using **React**
- Demonstrate **component-based architecture**
- Implement **state management using hooks**
- Enable **client-side routing**
- Design a **responsive and user-friendly UI**
- Simulate backend APIs using **JSON Server / static data**

---

Tech Stack
- **Frontend:** React.js  
- **Styling:** CSS 
- **State Management:** useState, useEffect  
- **Mock Backend:** JSON Server / Static JSON  
- **Version Control:** Git & GitHub  
- **Deployment:** 

---

Features
-  Authentication UI (Login / Signup)
-  Home feed with dynamic content
-  Search functionality
-  Like / Comment / Interaction features
-  Responsive design for all screen sizes
-  API simulation using JSON data
-  Smooth navigation with React Router
